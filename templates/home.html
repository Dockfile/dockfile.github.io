Dockerfile Project dockerfile.github.io

Dockerfile Project maintains a central repository of Dockerfile for various popular open source software services runnable on a Docker container. The goal of Dockerfile Project is to make it convenient for any developer to download pre-built Docker images and rapidly build, deploy awesome applications.
All trusted builds published to the public Docker Registry are automatically built from the projects hosted on Github, and they are kept in sync and protected by Docker Index.
Docker is an open source project to pack, ship and run any application as a lightweight container, sponsored by Docker Inc. under the apache 2.0 licence.
